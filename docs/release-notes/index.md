# Release Notes

## Moil Meeting Desktop Application v2.0

**Release Date:** january 22, 2026

We are pleased to announce the official launch of **Moil Meeting Desktop Application v2.0**. This major release delivers a redesigned multi-mode workflow, richer configuration controls, and a more flexible media pipeline for professional 360-degree meetings and surveillance.

---

### Overview

Moil Meeting v2.0 is a significant leap forward in usability and control. The application now emphasizes fast mode switching, advanced view configuration, and streamlined interactions so teams can operate complex meetings from a single interface.

### Highlights

-   **Multi-Mode Workflow:** Dedicated modes for Original, Discussion, Global, Patrol, and Presentation use cases.
-   **Flexible Media Sources:** Start with USB cameras, video files, IP cameras, or token-authenticated streams.
-   **Powerful View Controls:** Right-click actions and Anypoint transformation for precise view tuning.
-   **Unified Configuration:** Centralized panel for view labels, orientation, resolution, panorama, UI scale, and cache.

---

### What's New

#### Core Features

-   **Expanded Source Selection:** Choose **Open Camera**, **Load Media**, **Camera URL**, or **Token** when starting a session.
-   **Mode-First Navigation:** Bottom navigation optimized for rapid switching across operational modes.
-   **Discussion Mode Configuration:** Participant limits, view list controls, layout slider, and panorama toggle.
-   **Global & Patrol Enhancements:** Dual panoramic strips and automated scan controls.
-   **Presentation Focus:** Anypoint transformation tooling for accurate de-warped views.
-   **Right-Click Context Menu:** View info, snapshots, original view toggle, pause view, and configuration shortcuts.
-   **Configuration Panel:** Camera info and update checks, view labels, flip orientation, panorama config, resolution, UI scale, and cache management.
-   **Shortcut Support:** Keyboard shortcuts to assist users with **minimal mouse** usage.
-   **Multilingual Interface:** Support for multiple languages to enhance accessibility for a global user base.

### Improvements

-   **UI Consistency:** Cleaner layout and more predictable controls across all modes.
-   **Rendering Performance:** Smoother view updates and better responsiveness under load.
-   **Tracking Reliability:** Improved stability when mixing AI tracking with manual view controls.

### Bug Fixes

-   Stability fixes for configuration persistence and cache operations.
-   Reliability improvements for right-click actions and view pausing.
-   General fixes to reduce rendering glitches when switching modes.

---

### Upgrade Instructions

**No special actions are required.** Existing users can upgrade directly to v2.0. All user configurations and data will be preserved automatically, but we recommend reviewing the new configuration panel for updated controls.
